# Diretório de Imagens

Este diretório contém as imagens estáticas da aplicação.

## Logo do VitVet

A logo do VitVet deve ser salva como:
- **Nome do arquivo**: `vitvet-logo.png`
- **Localização**: `frontend/static/images/vitvet-logo.png`

A logo será exibida em:
1. **Sidebar (Menu lateral desktop)**: Substitui o ícone anterior
2. **Header (Cabeçalho)**: Aparece em todas as telas autenticadas, ao lado do título

### Especificações recomendadas:
- Formato: PNG (com transparência se necessário) ou SVG
- Tamanho recomendado: 180px de largura (será redimensionado automaticamente)
- Proporção: Manter proporção original da logo
